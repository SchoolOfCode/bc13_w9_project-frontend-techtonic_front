Class: SearchBar

SearchBar(value, className, handleChange, placeholder)

new SearchBar(value, className, handleChange, placeholder)

Parameters:
Name Type Description
value string

Is a string generated as the user types into the input field.
className string

A string for the class name.
handleChange func

A function that sets the SearchValue state to the e.target.value of the input field as the user types to search. compares the user input with the Resources state (set initially to contain all resources from database fetch request (see AddAndSearchDisplay Ln - 17)) returning any results in the Resources state that contain a partial match.
placeholder string

A string that sets the text displayed in the input before any the user types to search..
