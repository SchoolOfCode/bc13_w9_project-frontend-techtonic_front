Class: PopUp

PopUp(trigger, setTrigger)

new PopUp(trigger, setTrigger)

The PopUp Component contains a form that allows the user to add a resource to the database. It has an add button and a close button. It is triggered when the user clicks on 'Add a resource' button (see AddAndSearchDisplay ln-59)

Parameters:
Name Type Description
trigger boolean

This is a state variable is used in conditional rendering of the popup.
setTrigger func

This is the function used to set the value of trigger to true. (The default value is set to false)
